Pacotes para o ambiente virtual:

Antençao: Usar o comando <python -m venv "nome do ambiente"> para criar o ambiente virtual

Comando para ativar: source "nome do ambiente"/bin/activate

Comando de instalaçao: pip install customtkinter==5.2 opencv-python==4.5.3.56 Pillow==8.1.2 numpy==1.25.2 tflite_runtime==2.12.0 tflite_support==0.4.3
